# Network Science Analysis of LaLiga 2023-24

**Author:** Núria Pascual Salas

**Program:** MSc in Fundamental Principles of Data Science, Universitat de Barcelona

This project analyses the passing networks of the 2023-24 LaLiga season using network science to study what passing-network structure describes about a team and what it fails to predict about its performance.

## Overview

Each match is represented as a directed weighted graph where players are nodes and completed passes are edges. The project covers all 20 teams and 380 matches (760 team-match observations). 

The analysis has two parts:

- **Descriptive:** topological metrics, playing style (Switching Factor), eigenvector centrality, player-influence distribution (entropy, effective number of players), and structural fragility under targeted attacks.
- **Predictive:** whether these descriptors predict results at the team level (league points), the player level (match outcome from individual player data), and the match level, including a graph neural network.

## Data

The LaLiga 2023-24 event data is in `data/laliga.zip`. The notebooks read it directly from there.

## Notebooks

Run the notebooks sequentially, as each one uses CSV outputs generated by earlier notebooks.

- `01_switching_factor_analysis` - Switching Factor and SF-points correlation per team
- `02_eigenvector_centrality` - Eigenvector centrality per team
- `03_topological_metrics` - Topological metrics per team
- `04_player_centralities` - PageRank and centralities per match
- `05_robustness_analysis` - Structural fragility (PageRank attack)
- `05b_robustness_analysis_betweenness` - Structural fragility (betweenness attack)
- `05c_robustness_analysis_diameter_lcc` - Fragility (diameter and LCC)
- `06_player_match_matrix` - Player-match centrality matrix
- `07_fragility_vs_performance` - Fragility versus league points
- `07b_centrality_vs_fragility_check` - Check: fragility vs centrality concentration
- `08_multimetric_profile` - Multi-metric team profile
- `09_multivariate_team_level` - Team-level prediction of league points
- `10_bivariate_player_analysis` - Player centrality by position and outcome
- `11_logistic_regression` - Player-level logistic regression
- `12_gnn_match_prediction` - Graph Neural Network for match outcome
- `13_entropy_neff_analysis` - Season-level entropy and effective number of players
- `14_match_entropy_vs_result` - Match-level entropy versus result
- `15_centrality_distribution_shape` - Shape of the centrality distribution

## Code organisation

Shared functions for data loading, network construction, and network metrics are collected in `utils.py`. Functions specific to a single analysis are defined in the corresponding notebook.

## Outputs

Notebooks write CSV files to `outputs/csv/` and figures to `outputs/figures/`. Not all of them are included in the PDF of the thesis.
